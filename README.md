# laravel-chat

# setup

```
composer install
```
```
yarn install
```
```
php artisan package:discover --ansi
```
```
php -r "file_exists('.env') || copy('.env.example', '.env');"
```
```
php artisan key:generate --ansi
```
Изменить доступы к БД и ключи предоставленные pusher в файле .env
```
php artisan migrate
```
```
yarn prod
```
```
php artisan serve
```
